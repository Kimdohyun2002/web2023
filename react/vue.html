yarn

웹팩(Webpack)은 JavaScript 애플리케이션을 위한 정적 모듈 번들러입니다.

웹팩은 애플리케이션의 모든 모듈을 대상으로 의존성 그래프를 작성하고 이를 기반으로 번들(bundle)을 생성합니다.

이 번들은 여러 개의 JavaScript 파일과 관련 리소스들을 하나로 묶어서 브라우저에서 로드할 수 있는 최적화된 형태로 만들어 줍니다.

웹팩은 다양한 모듈 형식을 지원하며, 의존성 그래프를 생성하여 모듈 간의 종속성을 파악합니다. 이를 통해 필요한 모든 모듈을 찾아 하나의 번들로 묶고,

필요에 따라 변환(transform)과 최적화(optimization)를 수행합니다. 주로 웹 애플리케이션에서 사용되지만, 웹팩을 사용하여 JavaScript를 번들링할 수 있는 다양한 환경에서 활용할 수 있습니다.

yarn 설치

1.1 yarn cli 설치하기

뷰를 시작하기 전에 yarn cli를 설치해야 합니다. 

npm choco install yarn

1.2 vue cli 설치

npm install -g @vue/cli

1.3 파일 생성

vue create site-vue 를 작성하면 파일이 생성됩니다.

vue create site-vue

3번째 메뉴얼 선택

1.4 npm run serve

npm run serve을 작성하면 실행됩니다.

npm run serve
